重要発見: 家賃の見立てでプランの順位が入れ替わる
  • 見立てどおり（22万）: C > B > A → ハウジングセンターのシステムが有利
  • 控えめな見立て（+1.5万）: B > C ≒ A → 我が家はここを採用想定
  • 強めの見立て（+3万）: A > B > C → 自己資金が逆に最も有利

家賃が上振れるほどリノプロのプランは不利になる仕組み。これは 「家賃が上振れた取り分をハウジングセンターが取る契約」 だから。

2-3. なぜ「控えめな見立て」を採用するか

📊 客観的なデータ
  • 近隣事例の平均家賃: 19.06万円
  • 近隣事例の最高値: 21.00万円（東急ドエル・アルス）
  • ハウジングセンター見立て: 22.00万円 → 近隣最高値超え
  • リフォーム後の上乗せ分: 通常+1〜2万円

結論: 22万円が現実的な範囲で、リフォーム後にもう少し上振れる程度（23.5万円）を控えめな見立てとして採用。

💭 リスクの考え方

強めの見立て（25万円）もあり得るが、海外居住で市場調査の確かさに限界がある。控えめな見立てで計画を立て、上振れたら追加メリットと考えるのが心理的に健全。

逆に家賃が予想を下回るリスクもあるが、B（リノプロ）なら借上げ95%が5年間保証されるため、下振れリスクの大部分を回避できる。

3-2. 最大の隠れ利益: 借上げ家賃と実際の家賃の差額

ハウジングセンターの「相場22万円」見立ての根拠を、近隣事例から検証。

近隣物件の家賃事例 vs ハウジングセンター見立て（万円）
ハウジングセンターの見立てはリフォーム前提でも近隣最高値を超える強気
こう読み取れる ハウジングセンターが22万円で「貸せる」と見立てている以上、実態は 22〜25万円で貸せる可能性が高い。 しかし借上げ価格は95%/93%固定なので、家賃の上振れ分はすべてハウジングセンターの利益になる。 これが借上げビジネスの典型的な仕組み。

3-3. 表面 vs 実態 まとめ

ハウジングセンターの表面上の説明実態
「自己資金不要」 工事費に利益150-200万円が織り込み済み。相見積もりすれば50-100万円圧縮可能
「相場22万円で見立て」 近隣事例平均19万円。実際は22-25万円で貸せる可能性、上振れ分はハウジングセンターの利益
「空室保証あり」 実際に発動するケースは少なく、保証コストは名目的
「修繕費負担なし」 リノプロ工事範囲のみ。給湯器・配管などは家主負担
「手離れ運営」 家賃上昇の取り分を全てハウジングセンターに譲渡
冷静な評価 これは 詐欺ではなく借上げ業界の標準的な仕組み。 ハウジングセンターも適正な利益を取ってビジネスを成立させているだけ。 重要なのは「我が家にとって、この対価を払う価値があるか」を判断すること。次章で結論。

4-3. 工事を任せるメリット（ハウジングセンターに依頼する理由）

✓ ハウジングセンターに依頼するメリット
  • マンション管理組合との折衝に慣れている（着工3週間前の承認手続き）
  • 建材手配・職人手配の段取りが早い
  • 建物管理部の大塚さん体制でアフターサポート
  • 退去時の引き継ぎがスムーズ
  • 不具合発生時のクレーム窓口が明確
⚠ 他社に見積もりを取るリスク
  • 海外から業者選定・工事の進捗管理をする負担
  • 管理組合の折衝を一から実施
  • 知らない業者の品質リスク
  • 陽子さんに施工立会いを依頼する必要
  • 不具合時のクレーム対応も家主自身
結論: 工事はハウジングセンターに任せる 75万円圧縮できる可能性はあるが、海外からの工事管理の負担を考えると割に合わない。値引き交渉で30万円程度の譲歩を引き出せれば御の字。専門家の仕事の価値を評価する。
